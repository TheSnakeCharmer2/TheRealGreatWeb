Canvas in HTML

This is an example of a simple, blank canvas in HTML, which is an element used to draw graphics using <canvas> to create it:

The code for a simple canvas is fairly straight-forward.

<canvas id="myCanvas" width="200" height="100"
style="border:1px solid #000000;">
</canvas>

This is applied to all canvases but with more to go with it.

Canvases can be colored as well by using <style> along with the color coding as well (red would be #FF0000):

The border of a canvas can be colored alone:

The code is really simple for this red-outlined canvas.

<canvas id="myCanvas2" width="200" height="100" style="border:1px solid #FF0000;">
</canvas>

The whole canvas would be filled as well:

The coding for this filled canvas is quite self-explanatory when you look at it.

<canvas id="myCanvas3" width="200" height="100" style="border:1px solid #c3c3c3;">
</canvas>
<script>
var c = document.getElementById("myCanvas3");
var ctx = c.getContext("2d");
ctx.fillStyle = "#FF0000";
ctx.fillRect(0,0,200,100);
</script>

Simple Gradients can be made as well using Javascript to style it:

Linear gradients start from one end as one color, and fade to a different color towards the other side, they are created by writing "ctx.createLinearGradient". This one goes black to white from the left to the right, and direction is defined only from 0 to 1:

This definitely gets more complex, where the gradient starts and stops has to set along with the color. And the amount of space that the gradient of the canvas takes up can be changed.

<canvas id="myCanvas4" width="200" height="100" style="border:1px solid #d3d3d3;">
</canvas>
<script>
var c = document.getElementById("myCanvas4");
var ctx = c.getContext("2d");
// Create gradient
var grd = ctx.createLinearGradient(20,20,200,0);
grd.addColorStop(0,"black");
grd.addColorStop(1,"white");
// Fill with gradient
ctx.fillStyle = grd;
ctx.fillRect(0,0,200,100);
</script>

Radial gradients are a circular gradient, and they start in the designated spot that they are located.

The code for this is rather complex, because location of where the gradient start has to be set and how much space it takes in the canvas.

<canvas id="myCanvas5" width="200" height="100" style="border:1px solid #d3d3d3;">
</canvas>
<script>
var c = document.getElementById("myCanvas5");
var ctx = c.getContext("2d");
// Create gradient
var grd = ctx.createRadialGradient(75,50,5,90,60,100);
grd.addColorStop(0,"black");
grd.addColorStop(1,"white");
// Fill with gradient
ctx.fillStyle = grd;
ctx.fillRect(0,0,200,100);
</script>

There is a lot more that canvases can do, but these are the basics of what they are capable of.
