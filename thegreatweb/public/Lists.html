Below are examples of different lists in HTML

This is an example of an ordered list. Notice how the items are numerically listed:

  1. Item 1
  2. Item 2
  3. Item 3
  4. Item 4

The code for this numerical ordered list is rather simple:

<ol>
<li>Item 1</li>
<li>Item 2</li>
<li>Item 3</li>
<li>Item 4</li>
</ol>

This can also be done in alphebetical order. All that has to be done is start the ordered list with <ol type="A">, and then it will result in this below:

  1. Item 1
  2. Item 2
  3. Item 3
  4. Item 4

This is an example of an unordered list. Notice how the items are only listed by bullet points instead of numbers:

  • Item 1
  • Item 2
  • Item 3
  • Item 4

The coding here is very similar to the ordered list, the only thing that has changed is the <ol> at the beginning and the </ol> at the end. The starting tag is now <ul> and the ending tag is </ul>for an unordered list.

Here is an example of just a plain, description list:

Item 1
Item 2
Item 3
Item 4

The coding is set up the same as the other two lists, but this code has a different tag and title for each item.

<dl>
<dt>Item 1</dt>
<dt>Item 2</dt>
<dt>Item 3</dt>
<dt>Item 4</dt>
</dl>
