Div in HTML

Div allows different coloring on text for the body and paragraphs of HTML Pages.

Depending on which color coding you enter, the color of the text will be different.

It's up to you which colors you want to use when making a page.

A simple Div coding in Red looks like this:

<div style="color:#FF0000">

(Some words in red)


</div>

There are different colors that can be used, and each have different coding. Basic color codes are:
- Red=#FF0000
- Black=#000000
- Blue=#0000FF
- Green=#00FF00
- Yellow=#FFFF00
However, there are so many more codes for colors that I will not list all of them.
